Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element provides data for the browser that is not shown to the user.
  • Element tag commands are started with brackets and closed with a forward slash inside of the bracket.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how uch space we want around the content inside an element
  • I can use a class selector to apply CSS rules to all elements in the initial tag
  • I can use hex codes to apply a color to a CSS element
  • I can use "px" as a unit of measurement when creating a CSS feature
  • The order I can use for a CSS styling command is: Width, Margin, Padding, Border

Git

  • git checkout -b branch-name: creates a new branch and switches to it
